ensure_pdoinstance (   $instance = null)
Parameters
string$instanceIf supplied, this is the ID of the instance to use to execute the query. Optional, PDODEFINSTNAME constant by default.
Returns
string Instance ID used.

Ensures to provide a valid and connected instance of PDO, here are the steps: If it is not loaded, this function attempts to load the database configuration file. If not supplied as a parameter, this function attempts to determine an existing instance name. If the instance is not connected, this function attempts to connect.
